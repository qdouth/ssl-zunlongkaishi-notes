# ssl-zunlongkaishi-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及技术笔记、工具演示、静态原型等，彼此之间没有强关联，也不针对任何特定网站或服务。仓库的主要目的是提供一个集中存放和公开访问的场所，方便后续查阅与分享。

## 目录说明

仓库根目录下存放各 HTML 文件及其相关资源（如 CSS、JavaScript、图片等）。每个页面通常独立为一个子文件夹或单个文件，命名尽量反映页面主题。

```
ssl-zunlongkaishi-notes/
├── page-one/          # 示例：第一个页面文件夹
│   ├── index.html
│   └── assets/
├── page-two.html      # 示例：单个页面文件
└── ...
```

## 页面归档说明

- 所有页面均为纯前端静态 HTML，不依赖后端服务。
- 页面内容可能包含实验性、临时性或学习性质的笔记与演示。
- 部分页面可能引用外部资源（如 CDN 库），但不会收集用户数据或执行跟踪。
- 归档内容会不定期更新或清理，不保证所有页面长期有效。

## 维护说明

- 本仓库由个人维护，主要用于个人笔记与学习记录。
- 欢迎通过 Issues 指出页面中的错误或提出改进建议。
- 不接受直接向仓库提交涉及无关主题、商业推广或违反社区准则的内容。
- 仓库可能随时调整目录结构或删除旧页面，恕不另行通知。

## 使用方式

你可以直接通过 GitHub Pages 或本地浏览器打开仓库中的 HTML 文件。若使用 GitHub Pages，请确认仓库设置中已启用 Pages 功能，并选择正确的源分支（通常为 `main` 或 `gh-pages`）。
